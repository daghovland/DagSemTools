# OWL 2 RL 2 Datalog

This project is a simple translation from OWL 2 RL to Datalog, mainly a naive translation of
the rules in https://www.w3.org/TR/owl2-profiles/#Reasoning_in_OWL_2_RL_and_RDF_Graphs_using_Rules
into datalog.
