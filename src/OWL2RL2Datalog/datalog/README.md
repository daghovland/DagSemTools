# OWL 2 RL Axiomatization
This folder contains some of the rules in https://www.w3.org/TR/owl2-profiles/#Reasoning_in_OWL_2_RL_and_RDF_Graphs_using_Rules and https://www.w3.org/TR/owl2-profiles/
